Accessibility

Accessibility statement for My Own Fable.

We want My Own Fable to be easy to read and navigate for as many visitors as possible. The site uses responsive layouts, semantic HTML, visible focus states, and readable contrast choices. A skip link is included to help keyboard and assistive technology users move directly to main content.

We continue to improve accessibility over time. If you encounter an accessibility barrier, please contact us and describe the page and issue. Helpful feedback allows us to make the site better for everyone.
